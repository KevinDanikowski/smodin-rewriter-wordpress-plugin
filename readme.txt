Smodin rewriter plugin is meant to be used with the smodin rewriter API available at https://rapidapi.com/smodin/api/rewriter-paraphraser-text-changer-multi-language. The plugin allows one to rewrite text in various languages for wordpress posts one at a time.

== Description ==

Smodin rewriter can be used with the languages below to rewrite one post at a time. You can send HTML which works without issue in majority of cases, however at times the HTML can break using the rewriter API. Some languages are likely to break the HTML more than others.

**Localized in 30 Languages**

A few of our supported languages:

* English
* Spanish
* German
* French
* Portuguese
* Italian
* indonesian
* Swedish
* Dutch
* Arabic
* Turkish
* catalan
* Polish
* malay
* philipino
* Danish
* Ukranian
* Romanian
* Norwegian
* Hungarian
* Chinese
