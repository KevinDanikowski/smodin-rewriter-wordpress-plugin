Smodin rewriter plugin requires an account with the Smodin Rewriter API at https://rapidapi.com/smodin/api/rewriter-paraphraser-text-changer-multi-language. The plugin allows one to rewrite text in various languages for wordpress posts one at a time. It worked by calling the Smodin Rewriter API.

== Description ==

Smodin rewriter plugin allows the use of the Smodin Rewriter API (available at https://rapidapi.com/smodin/api/rewriter-paraphraser-text-changer-multi-language) which can be used to rewrite posts, one at a time, in various languages such as the ones below. The plugin works by calling the Rapid API endpoint associated with the Smodin Rewriter API. You can send HTML which works without issue in majority of cases, however at times the HTML can break using the rewriter API. Some languages are likely to break the HTML more than others. Lastly, after installation, make sure to choose the default language that your plugin will be rewriting.

== Installation ==

1. Search for Smodin Rewriter in your WordPress backend and click install, or download the smodin-rewriter.zip file and unzip it.
2. If you downloaded the zip, upload the smodin-rewriter folder into wp-content/plugins folder
3. Activate the plugin in your WordPress Admin area.
4. Go to "Settings" to add the API key (You can acquire a free API key here https://rapidapi.com/smodin/api/rewriter-paraphraser-text-changer-multi-language)
5. Go to a post to rewrite
